rtmdds
======

DDS transport for OpenRTM-aist.

This is the RC2 release of a simple implementation of a port for using DDS
transports in OpenRTM-aist RT-Components. It is a preliminary implementation
for evaluation purposes. Future versions may change the API considerably.

This release supports DDS implementations from RTI, OpenDDS, and OpenSplice.
