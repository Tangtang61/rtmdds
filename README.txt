rtmdds
======

DDS transport for OpenRTM-aist.

This is the RC1 release of a prototype implementation of a port for
using DDS transports in OpenRTM-aist RT-Components. It is a preliminary
implementation for evaluation purposes. Future versions may change the
API considerably.
